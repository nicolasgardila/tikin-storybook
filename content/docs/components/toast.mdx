```tsx title="toast-usage.tsx"
export function Toast({ message, variant = 'info', position = 'top-right' }) {
  return (
    <div className={cn(
      "fixed z-50 flex items-center gap-3 rounded-xl bg-[#0B0B0B] px-4 py-3 text-sm text-white shadow-md",
      position === 'top-right' && "top-4 right-4 max-w-[360px]",
      position === 'bottom-center' && "bottom-4 left-4 right-4"
    )}>
      <ToastIcon variant={variant} />
      <span>{message}</span>
    </div>
  );
}
```

## Guias de Uso

<DoDont type="do">
  Usa Toast para feedback no critico: acciones completadas, datos guardados, elementos copiados.
</DoDont>

<DoDont type="dont">
  No uses Toast para errores que requieren accion del usuario. Usa Dialog para esos casos.
</DoDont>

<DoDont type="do">
  Manten los mensajes cortos y claros — una linea maximo. El toast debe leerse en 2 segundos.
</DoDont>

<DoDont type="dont">
  No muestres mas de un toast a la vez. Usa una cola que muestre uno por uno.
</DoDont>
