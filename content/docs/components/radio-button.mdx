```tsx title="radio-group.tsx"
export function RadioGroup() {
  const [selected, setSelected] = useState('transfer');

  const options = [
    { value: 'transfer', label: 'Transferencia bancaria' },
    { value: 'debit', label: 'Tarjeta de debito' },
    { value: 'cash', label: 'Efectivo' },
  ];

  return (
    <div className="flex flex-col gap-1">
      {options.map((option) => (
        <label
          key={option.value}
          className="flex items-center gap-2 py-1.5 px-2.5 pl-1.5 rounded-lg cursor-pointer text-sm font-['Funnel_Display'] text-[#1E1E1E]"
          style={{ fontWeight: selected === option.value ? 600 : 400 }}
        >
          <span
            className={`w-5 h-5 rounded-full border-[1.5px] inline-flex items-center justify-center shrink-0 ${
              selected === option.value
                ? 'border-[#0B0B0B] bg-[#0B0B0B]'
                : 'border-[#D9D9D9] bg-white'
            }`}
          >
            {selected === option.value && (
              <span className="w-2 h-2 rounded-full bg-white" />
            )}
          </span>
          {option.label}
        </label>
      ))}
    </div>
  );
}
```

### Interacciones

El componente incluye feedback tactil al presionar: escala sutil (`scale 0.97`) con reduccion de opacidad, y un release con transicion spring natural. Este efecto se desactiva automaticamente en estados `disabled` y `loading`.

## Guias de Uso

<DoDont type="do">
  Usar radio buttons para seleccion unica entre opciones mutuamente excluyentes.
</DoDont>

<DoDont type="dont">
  No usar radio buttons si el usuario puede seleccionar multiples opciones. Usar Checkbox en su lugar.
</DoDont>
