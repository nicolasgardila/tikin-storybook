### Interacciones

El componente incluye feedback tactil al presionar: escala sutil (`scale 0.97`) con reduccion de opacidad, y un release con transicion spring natural. Este efecto se desactiva automaticamente en estados `disabled` y `loading`.

## Guías de Uso

<DoDont type="do">
  Usa Checkbox para opciones independientes donde el usuario puede activar o desactivar cada una sin afectar las demás.
</DoDont>

<DoDont type="dont">
  No uses Checkbox para selección exclusiva (donde solo una opción puede estar activa). Usa Radio Buttons en ese caso.
</DoDont>

<DoDont type="do">
  Incluye siempre un label descriptivo junto al checkbox. El área de click debe incluir el label completo.
</DoDont>

<DoDont type="dont">
  No uses un checkbox sin label visible. Un checkbox aislado no comunica su propósito al usuario.
</DoDont>
