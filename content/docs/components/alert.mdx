## Guias de Uso

<DoDont type="do">
  Usar el color y el icono apropiado para cada tipo de mensaje. Info para contexto, Success para confirmaciones, Warning para precauciones, Error para fallas.
</DoDont>

<DoDont type="dont">
  No usar alerts para informacion de flujo normal. Reservarlos para estados excepcionales que requieren atencion del usuario.
</DoDont>
