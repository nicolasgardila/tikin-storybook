## Instalacion

```bash title="Terminal"
npm install @dstikin/ds
```

```tsx
import { Select } from '@dstikin/ds'
```

## Uso

```tsx title="select.tsx"
import { Select } from '@dstikin/ds'

export default function Example() {
  return (
    <Select
      placeholder="Seleccionar..."
      options={[
        { value: '1', label: 'Opción 1' },
        { value: '2', label: 'Opción 2' },
      ]}
    />
  )
}
```

## Guías de Uso

<DoDont type="do">
  Usa Select cuando hay entre 4 y 15 opciones. Para menos opciones, considera Radio Buttons. Para más, considera un campo con búsqueda.
</DoDont>

<DoDont type="dont">
  No uses Select para opciones binarias (sí/no). Usa un Checkbox o Switch en su lugar.
</DoDont>

<DoDont type="do">
  Incluye siempre una opción placeholder como "Seleccionar..." para indicar que no se ha elegido nada.
</DoDont>

<DoDont type="dont">
  No pre-selecciones una opción a menos que sea un valor por defecto genuino. El usuario debe tomar la decisión conscientemente.
</DoDont>
