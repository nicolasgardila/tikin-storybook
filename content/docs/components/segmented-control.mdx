```tsx title="segmented-control-usage.tsx"
export function SegmentedControl() {
  const [activeIndex, setActiveIndex] = useState(0);
  const segments = ['Todo', 'Ingresos', 'Gastos'];

  return (
    <div className="inline-flex items-center bg-[#F6F6F6] rounded-xl p-1 gap-0.5">
      {segments.map((segment, i) => (
        <button
          key={segment}
          onClick={() => setActiveIndex(i)}
          className={`rounded-lg px-5 py-2 text-sm font-medium border-none cursor-pointer h-9 flex items-center transition-all duration-200 ${
            activeIndex === i
              ? 'bg-white text-[#0B0B0B] shadow-sm'
              : 'bg-transparent text-[#999999]'
          }`}
        >
          {segment}
        </button>
      ))}
    </div>
  );
}
```

### Interacciones

El componente incluye feedback tactil al presionar: escala sutil (`scale 0.97`) con reduccion de opacidad, y un release con transicion spring natural. Este efecto se desactiva automaticamente en estados `disabled` y `loading`.

## Guias de Uso

<DoDont type="do">
  Usar para alternar entre 2 a 4 vistas o modos dentro del mismo contexto.
</DoDont>

<DoDont type="dont">
  No usar para mas de 5 opciones. Considerar Tabs o Select en su lugar.
</DoDont>
