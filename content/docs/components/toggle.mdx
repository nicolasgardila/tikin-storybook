## Instalacion

```tsx
import { DSToggle } from '@/components/ds/ds-toggle'
```

## Uso

```tsx title="toggle.tsx"
import { DSToggle } from '@/components/ds/ds-toggle'

export default function Example() {
  return (
    <DSToggle
      checked={false}
      label="Notificaciones push"
      onChange={(checked) => console.log(checked)}
    />
  )
}
```

### Interacciones

El componente incluye feedback tactil al presionar: escala sutil (`scale 0.97`) con reduccion de opacidad, y un release con transicion spring natural. Este efecto se desactiva automaticamente en estados `disabled` y `loading`.

## Anatomia

Estructura del Toggle switch:

```
┌──────────────────────────────────────┐
│  Label Text (Ubuntu Sans, 16px)      │
│  gap: 12px                           │
│                                      │
│  Track: 44px × 24px                  │
│  ┌────────────────────────────────┐  │
│  │ ┌────┐                        │  │
│  │ │thumb│  ←→  2px inset        │  │
│  │ │20×20│                        │  │
│  │ └────┘                        │  │
│  └────────────────────────────────┘  │
│  border-radius: 12px (track)         │
│  border-radius: 10px (thumb)         │
│  transition: 200ms ease              │
└──────────────────────────────────────┘
```

| Parte | Valor |
|-------|-------|
| Track | 44 x 24px, r12 |
| Thumb | 20 x 20px, r10 |
| Inset | 2px from track edge |
| Label gap | 12px |
| Off color | #DEDEDE (track), #FFFFFF (thumb) |
| On color | #0B0B0B (track), #FFFFFF (thumb) |

## Guias de Uso

<DoDont type="do">
  Usa Toggle para configuraciones binarias de activacion/desactivacion como notificaciones, modo oscuro, o preferencias.
</DoDont>

<DoDont type="dont">
  No uses Toggle para acciones que requieren confirmacion. Usa un Button con dialogo de confirmacion en su lugar.
</DoDont>

<DoDont type="do">
  Siempre incluye un label descriptivo junto al toggle para que el usuario entienda que esta activando.
</DoDont>

<DoDont type="dont">
  No uses Toggle para seleccion entre multiples opciones. Usa Radio Buttons o un Select en su lugar.
</DoDont>
