## Instalacion

```tsx
import { DSButton } from '@/components/ds/ds-button'
```

## Uso

```tsx title="button.tsx"
import { DSButton } from '@/components/ds/ds-button'

export default function Example() {
  return (
    <DSButton variant="primary" size="md">
      Continuar
    </DSButton>
  )
}
```

### Interacciones

El componente incluye feedback tactil al presionar: escala sutil (`scale 0.97`) con reduccion de opacidad, y un release con transicion spring natural. Este efecto se desactiva automaticamente en estados `disabled` y `loading`.

## Anatomia

Estructura interna del boton Primary:

```
┌─────────────────────────────────────┐
│           padding-y: 8px            │
│  ┌─────────────────────────────┐    │
│  │  [Icon]  LABEL TEXT  [Icon] │    │
│  │   20px    14px/500    20px  │    │
│  │         gap: 8px           │    │
│  └─────────────────────────────┘    │
│           padding-x: 20px           │
│         border-radius: 12px         │
│      font: Ubuntu Sans Mono         │
│     transform: uppercase            │
│     tracking: 0.05em                │
│         height: 40px (md)           │
└─────────────────────────────────────┘
```

| Parte | Token | Valor |
|-------|-------|-------|
| Padding horizontal | `--ds-space-5` | 20px |
| Padding vertical | `--ds-space-2` | 8px |
| Border radius | `--ds-radius-md` | 12px |
| Icon size | — | 20px |
| Gap (icon-label) | `--ds-space-2` | 8px |
| Font | Ubuntu Sans Mono | 14px / 500 |
| Height (md) | `--ds-height-md` | 40px |

## Guías de Uso

<DoDont type="do">
  Usa el botón Primary para la acción principal de cada pantalla. Solo debe haber un Primary visible por contexto.
</DoDont>

<DoDont type="dont">
  No uses múltiples botones Primary en la misma vista. Usa Secondary o Ghost para acciones alternativas.
</DoDont>

<DoDont type="do">
  Usa texto uppercase en Ubuntu Sans Mono para los labels de botón, siguiendo el patrón tipográfico del DS.
</DoDont>

<DoDont type="dont">
  No uses sentence case o lowercase para labels de botón. El DS usa uppercase con tracking amplio para CTAs.
</DoDont>
