## Instalacion

```tsx
import { DSDialog } from '@/components/ds/ds-dialog'
```

## Uso

```tsx title="dialog.tsx"
import { DSDialog } from '@/components/ds/ds-dialog'

export default function Example() {
  return (
    <DSDialog
      open={true}
      onClose={() => {}}
      title="Confirmar acción"
    >
      <p>¿Estás seguro de que deseas continuar?</p>
    </DSDialog>
  )
}
```

### Estructura Double-Layer

```tsx title="dialog.tsx"
export function Dialog({ open, onClose, title, children }) {
  if (!open) return null;

  return (
    {/* Backdrop */}
    <div className="fixed inset-0 z-50 flex items-center justify-center bg-black/40">
      {/* Outer shell — double-layer */}
      <div className="rounded-3xl border border-[#DEDEDE] bg-[#F6F6F6] p-3 shadow-[0_0_2px_rgba(11,15,16,0.12)]">
        {/* Inner panel */}
        <div className="rounded-[1.25rem] bg-white p-9">
          <h3 className="font-funnel-display text-2xl font-normal">
            {title}
          </h3>
          {children}
        </div>
      </div>
    </div>
  );
}
```

## Guías de Uso

<DoDont type="do">
  Usa Dialog para confirmaciones y advertencias de acciones destructivas que requieren atención inmediata del usuario.
</DoDont>

<DoDont type="dont">
  No uses Dialog para mostrar contenido largo o formularios complejos. Usa Side Panel en su lugar.
</DoDont>

<DoDont type="do">
  Incluye siempre una acción para cerrar el dialog — ya sea un botón "Cancelar" o el botón X.
</DoDont>

<DoDont type="dont">
  No apiles múltiples dialogs uno encima de otro. Simplifica el flujo para evitar modals anidados.
</DoDont>
