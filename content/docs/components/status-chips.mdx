## Guias de Uso

<DoDont type="do">
  Usa Status Chips para indicadores de estado en tablas, listas y cards: Completado, Pendiente, Rechazado, En proceso.
</DoDont>

<DoDont type="dont">
  No uses Status Chips para acciones interactivas. Usa Buttons en su lugar.
</DoDont>

<DoDont type="do">
  Mantene consistencia en el uso de variantes: success para estados positivos, warning para atencion, error para negativos.
</DoDont>

<DoDont type="dont">
  No mezcles variantes arbitrariamente. Cada color tiene un significado semantico que debe respetarse en toda la app.
</DoDont>
