## Instalacion

```tsx

```

## Uso

```tsx title="container.tsx"

  return (
    <DSContainer variant="double" padding="default">
      <h3>Confirmar Transferencia</h3>
      <p>Monto: $1,200.00</p>
    </DSContainer>
  )
}
```

## Anatomía del Double-Layer

```
+-- Shell exterior (r24, p12) ------------------+
|  bg: #F6F6F6 | border: 1px #DEDEDE            |
|                                                |
|  +-- Panel interior (r20, p24/p36) ---------+  |
|  |  bg: #FFFFFF                              |  |
|  |                                           |  |
|  |  Contenido aquí                           |  |
|  |                                           |  |
|  +-------------------------------------------+  |
|                                                |
+------------------------------------------------+
```

## Especificaciones

| Propiedad | Neutral | Warm | Dark |
|-----------|---------|------|------|
| Shell bg | `#F6F6F6` | `#F1EFEB` | `#171717` |
| Shell border | `1px #DEDEDE` | `1px #E8E4DD` | `1px #2A2A2A` |
| Shell radius | 24px | 24px | 24px |
| Shell padding | 12px | 12px | 12px |
| Panel bg | `#FFFFFF` | `#FFFFFF` | `#1E1E1E` |
| Panel radius | 20px | 20px | 20px |
| Panel padding | 24px (panel) / 36px (dialog) | 24px / 36px | 24px / 36px |
| Shadow (neutral) | `0 0 2px rgba(11,15,16,0.12)` | — | `0 0 4px rgba(0,0,0,0.30)` |

## Guías de Uso

<DoDont type="do">
  Usa el double-layer para dialogs, panels y áreas de contenido complejo que necesitan separarse visualmente del fondo.
</DoDont>

<DoDont type="dont">
  No anides contenedores double-layer. Un solo nivel de doble capa es suficiente — anidarlos crea demasiada profundidad visual.
</DoDont>

<DoDont type="do">
  Usa la variante warm para contenedores en contextos financieros (wallet, cuentas, transacciones) y neutral para UI genérica.
</DoDont>

<DoDont type="dont">
  No uses el contenedor simple cuando necesitas separación visual clara del fondo. El double-layer comunica mejor la jerarquía de contenido.
</DoDont>
