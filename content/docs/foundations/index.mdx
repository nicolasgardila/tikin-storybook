---
title: Foundations
description: Tokens de diseno fundamentales — colores, tipografia, espaciado, radios, sombras
---

Los foundations definen los tokens base del sistema de diseno. Cada token tiene un valor exacto utilizado en produccion.

- [Colores](/docs/foundations/colors) — Sistema dual warm/neutral
- [Tipografia](/docs/foundations/typography) — Funnel Display, Ubuntu Sans, Ubuntu Sans Mono
- [Espaciado](/docs/foundations/spacing) — 19 tokens de 0 a 64px
- [Radio de Bordes](/docs/foundations/radius) — 10 tokens de radio
- [Sombras](/docs/foundations/shadows) — 5 niveles de elevacion
- [Bordes](/docs/foundations/borders) — 4 anchos de borde
- [Alturas](/docs/foundations/heights) — Tokens de altura de componentes
- [Grillas](/docs/foundations/grid) — Sistema 12/8/4 columnas
- [Corner Smoothing](/docs/foundations/corner-smoothing) — Suavizado 60%
