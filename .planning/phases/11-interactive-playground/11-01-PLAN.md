  <files>components/docs/playground-controls.tsx, components/docs/component-playground.tsx</files>
  <action>
    **1. Create `components/docs/playground-controls.tsx`**

    This is NOT a client component — it's a shared types + control renderer module.

    Define the control types and render functions:

    ```tsx
    'use client';

    import { useState } from 'react';

    // Prop control definition types
    export type ControlType = 'select' | 'boolean' | 'text' | 'number';

    export interface PropControl {
      name: string;
      type: ControlType;
      defaultValue: any;
      options?: string[];  // for 'select' type
      label?: string;      // display label (defaults to name)
      min?: number;         // for 'number' type
      max?: number;         // for 'number' type
    }

    interface ControlRowProps {
      control: PropControl;
      value: any;
      onChange: (name: string, value: any) => void;
    }
    ```

    Render a single control row:
    - **select**: `<select>` dropdown with options
    - **boolean**: toggle switch (reuse toggle pattern from DS)
    - **text**: text input
    - **number**: number input with optional min/max

    Each row shows: label on left, control on right, in a horizontal layout.
    Style with --docs-* variables for the controls panel chrome.

    Export `ControlRow` component and `PropControl` type.

    **2. Create `components/docs/component-playground.tsx`**

    Client component that combines:
    - Preview area (top) — renders the component with current prop values
    - Controls panel (bottom) — renders controls for each prop

    ```tsx
    'use client';

    import { useState, type ReactNode, type ComponentType } from 'react';
    import { ControlRow, type PropControl } from './playground-controls';

    interface ComponentPlaygroundProps {
      component: ComponentType<any>;
      controls: PropControl[];
      children?: ReactNode;  // optional static content to show alongside
    }
    ```

    The playground:
    1. Initializes state from control defaultValues
    2. Renders the component with current state as props
    3. Has viewport (desktop/mobile) and theme (light/dark) toggles like ComponentPreview
    4. Below the preview, renders a controls panel with a row for each PropControl
    5. When any control changes, updates state → component re-renders live

    Layout:
    ```
    ┌─────────────────────────────────────────┐
    │ [Desktop] [Mobile]           [🌓 theme] │  ← toolbar
    ├─────────────────────────────────────────┤
    │                                         │
    │         Live Component Preview          │  ← preview area (same as ComponentPreview)
    │                                         │
    ├─────────────────────────────────────────┤
    │ Controls                          (6)   │  ← controls header with count
    ├─────────────────────────────────────────┤
    │ variant    │ [primary ▼]                │  ← select control
    │ size       │ [md ▼]                     │  ← select control
    │ disabled   │ [○ toggle]                 │  ← boolean control
    │ fullWidth  │ [○ toggle]                 │  ← boolean control
    │ children   │ [Continuar]                │  ← text control
    └─────────────────────────────────────────┘
    ```

    The preview area reuses the same CSS variable pattern as ComponentPreview (--preview-brand, --preview-surface, etc.).

    The controls panel uses --docs-* variables for styling (it's part of the docs chrome, not the DS preview).

    Controls panel style:
    - Background: var(--docs-bg)
    - Border-top: 1px solid var(--docs-border)
    - Each row: border-bottom 1px solid var(--docs-border), padding 8px 12px
    - Label: font-size 13px, font-weight 500, color var(--docs-fg)
    - Controls: font-size 13px, appropriate input styling

    IMPORTANT: The `component` prop receives a React component reference, NOT a string. The playground renders `<Component {...currentProps} />` directly.

    **1. Create `components/ds/ds-button.tsx`**

    A real React component implementing the DS button with all variants:

    ```tsx

    interface DSButtonProps {
      variant?: 'primary' | 'secondary' | 'ghost';
      size?: 'sm' | 'md' | 'lg';
      disabled?: boolean;
      fullWidth?: boolean;
      children?: ReactNode;
    }
    ```

    Implementation:
    - Uses inline styles (not Tailwind) so it works in the playground preview context
    - Reads --preview-brand/--preview-brand-on for dark mode adaptation
    - Primary: bg var(--preview-brand), text var(--preview-brand-on), rounded 12px
    - Secondary: bg transparent, border 1px var(--preview-border), text inherit
    - Ghost: bg transparent, no border, text inherit
    - Size sm: py 8px, px 16px, font 12px
    - Size md: py 12px, px 24px, font 14px
    - Size lg: py 16px, px 32px, font 16px
    - Font: Ubuntu Sans Mono (var(--font-ubuntu-sans-mono)), uppercase, tracking 0.05em
    - Disabled: opacity 0.5, cursor not-allowed
    - fullWidth: width 100%

    This is NOT a 'use client' component — it's a pure presentational component that receives props.

    **2. Create a Buttons playground wrapper component**

    Create `components/ds/playgrounds/buttons-playground.tsx` as a client component:

    ```tsx
    'use client';

    import { ComponentPlayground } from '@/components/docs/component-playground';
    import { DSButton } from '@/components/ds/ds-button';
    import type { PropControl } from '@/components/docs/playground-controls';

    const buttonControls: PropControl[] = [
      {
        name: 'variant',
        type: 'select',
        defaultValue: 'primary',
        options: ['primary', 'secondary', 'ghost'],
      },
      {
        name: 'size',
        type: 'select',
        defaultValue: 'md',
        options: ['sm', 'md', 'lg'],
      },
      {
        name: 'disabled',
        type: 'boolean',
        defaultValue: false,
      },
      {
        name: 'fullWidth',
        type: 'boolean',
        defaultValue: false,
      },
      {
        name: 'children',
        type: 'text',
        defaultValue: 'Continuar',
        label: 'label',
      },
    ];

    export function ButtonsPlayground() {
      return <ComponentPlayground component={DSButton} controls={buttonControls} />;
    }
    ```
