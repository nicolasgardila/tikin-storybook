    Create `components/docs/type-specimen.tsx` as a client component that renders live font specimens.

    The component should:
    1. Be a `'use client'` component (for interactive font/weight selection)
    2. Accept no props — hardcode the DS type system data
    3. Show three font sections: Funnel Display, Ubuntu Sans, Ubuntu Sans Mono
    4. For each font, render a specimen table/grid showing:
       - Each size from the type scale (xs 12px through 5xl 48px)
       - The actual rendered text in that font at that size
       - Size label, px value, and line-height
    5. Use a sample text that demonstrates each font's character:
       - Funnel Display: "Bienvenido a Black Cash" (display text)
       - Ubuntu Sans: "El sistema de diseno para tu producto fintech" (body text)
       - Ubuntu Sans Mono: "$12,450.00" (numeric/CTA text)
    6. Allow toggling between font weights available for each font:
       - Funnel Display: Regular (400), Bold (700)
       - Ubuntu Sans: Regular (400), Medium (500), Semi-bold (600), Bold (700)
       - Ubuntu Sans Mono: Regular (400), Medium (500), Bold (700)
    7. Use a simple weight selector (buttons or segmented control) per font section

    Type scale data:
    ```ts
    const typeScale = [

      { name: 'lg', size: '18px', lineHeight: '28px' },
      { name: 'xl', size: '20px', lineHeight: '28px' },
      { name: '2xl', size: '24px', lineHeight: '32px' },
      { name: '3xl', size: '30px', lineHeight: '36px' },
      { name: '4xl', size: '36px', lineHeight: '40px' },
      { name: '5xl', size: '48px', lineHeight: '48px' },
    ];
    ```

    Font data:
    ```ts
    const fonts = [

      {
        name: 'Ubuntu Sans',

        tailwind: 'font-ds-body',
        role: 'Texto de cuerpo',
        sampleText: 'El sistema de diseño para tu producto fintech',
        weights: [
          { name: 'Regular', value: 400 },
          { name: 'Medium', value: 500 },
          { name: 'Semi-bold', value: 600 },
          { name: 'Bold', value: 700 },
        ],
      },
      {
        name: 'Ubuntu Sans Mono',
        cssVar: 'var(--font-ubuntu-sans-mono)',
        tailwind: 'font-ds-mono',
        role: 'Numeros y CTAs',
        sampleText: '$12,450.00',
        weights: [
          { name: 'Regular', value: 400 },
          { name: 'Medium', value: 500 },
          { name: 'Bold', value: 700 },
        ],
      },
    ];
    ```

    - Font name heading with role description
    - Weight selector (horizontal pill buttons, active state with --docs-* vars)
