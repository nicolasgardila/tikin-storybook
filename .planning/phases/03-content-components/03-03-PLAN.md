    ## Buttons

    El sistema incluye 7 variantes de botón: Primary, Secondary, Icon Primary, Icon Secondary, Icon Muted, Ghost y Link.

    ### Primary Button

    El botón principal para acciones primarias. Fondo oscuro con texto claro.

    ```tsx title="primary-button.tsx"
    export function PrimaryButton() {
      return (
        <button className="rounded-xl bg-[#0B0B0B] px-6 py-3 text-sm font-medium uppercase tracking-wider text-white">
          Continuar
        </button>
      );
    }
    ```

    ### Secondary Button

    Para acciones secundarias. Borde sutil con fondo transparente.

    ```tsx title="secondary-button.tsx"
    export function SecondaryButton() {
      return (
        <button className="rounded-xl border border-[#DEDEDE] bg-transparent px-6 py-3 text-sm font-medium uppercase tracking-wider text-[#0B0B0B]">
          Cancelar
        </button>
      );
    }
    ```

    ### Ghost Button

    Para acciones terciarias. Sin borde ni fondo.

    ```tsx title="ghost-button.tsx"
    export function GhostButton() {
      return (
        <button className="rounded-xl px-6 py-3 text-sm font-medium text-[#0B0B0B] hover:bg-[#F6F6F6]">
          Ver más
        </button>
      );
    }
    ```

    ## Props

      props={[

        {

          required: true,
        },
      ]}
    />

    ## Guías de Uso

    <DoDont type="do">
      Usa el botón Primary para la acción principal de cada pantalla. Solo debe haber un Primary visible por contexto.
    </DoDont>

    </DoDont>

    <DoDont type="dont">
