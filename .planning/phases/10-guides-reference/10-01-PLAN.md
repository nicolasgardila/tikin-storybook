  <name>Task 1: Create IconSearch component and icons reference page</name>
  <files>components/docs/icon-search.tsx, content/docs/guides/icons.mdx, mdx-components.tsx</files>
  <action>
    1. Create `components/docs/icon-search.tsx` as a client component:

       The component should:
       - Be a `'use client'` component
       - Accept no props — hardcode a curated list of ~40-50 commonly used Phosphor Icons
       - Render a search input at the top that filters icons by name
       - Display icons in a responsive grid (4-6 columns)
       - Each icon cell shows: the SVG icon (24x24), the icon name below
       - On click, copy the icon's SVG markup to clipboard with "Copiado!" feedback
       - Use --docs-* variables for chrome styling

       Icon data structure — define inline SVG paths for common Phosphor icons:
       ```ts
       const icons = [
         { name: 'House', svg: '<svg>...</svg>' },
         { name: 'MagnifyingGlass', svg: '<svg>...</svg>' },
         { name: 'Bell', svg: '<svg>...</svg>' },
         { name: 'User', svg: '<svg>...</svg>' },
         { name: 'Gear', svg: '<svg>...</svg>' },
         // ... ~40-50 icons total
       ];
       ```

       Include these common categories of Phosphor icons:
       - **Navigation**: House, ArrowLeft, ArrowRight, CaretDown, CaretUp, CaretRight, X, List (hamburger)
       - **Actions**: Plus, Minus, Pencil, Trash, Copy, Download, Upload, Share, Check
       - **Communication**: Bell, ChatCircle, Envelope, Phone
       - **User**: User, UserCircle, Users, SignOut, SignIn
       - **Content**: Eye, EyeSlash, Star, Heart, Bookmark, Flag
       - **System**: Gear, Info, Warning, XCircle, CheckCircle, Clock, Calendar
       - **Finance**: CreditCard, Bank, CurrencyDollar, Wallet, Receipt, ArrowsLeftRight
       - **Media**: Camera, Image, File, Folder

       For each icon, use the Phosphor Icons regular weight SVG (stroke style, not filled).
       All SVGs should be 24x24, viewBox="0 0 256 256", stroke="currentColor" or fill="currentColor" depending on Phosphor's convention.

       IMPORTANT: Since we can't dynamically load Phosphor icons at build time without adding the dependency, hardcode SVG paths for the ~40-50 most used icons. This keeps the docs site dependency-free.

       Search: Filter icons by name using a simple includes() check. Show count of matching results.

    2. Register in mdx-components.tsx:
       - Import: `import { IconSearch } from '@/components/docs/icon-search';`
       - Add to return: `IconSearch,`

    3. Create `content/docs/guides/icons.mdx`:
       ```
       ---
       title: Iconografia
       description: Referencia de iconos Phosphor usados en el Design System
       ---
       ```
       Content:
       1. Intro — Black Cash DS uses Phosphor Icons (phosphoricons.com) as the official icon library
       2. Usage guidelines:
          - Always use inline SVGs, never icon fonts
          - Default size: 24×24px (can scale to 16, 20, 32)
          - Color: inherits from parent text color via currentColor
          - Weight: Regular (default), Bold for emphasis
       3. `<IconSearch />` component
       4. Installation note — link to phosphoricons.com for the full library
       5. Code example showing icon usage:
          ```tsx
          <svg width="24" height="24" viewBox="0 0 256 256" fill="none" stroke="currentColor" strokeWidth="16">
            {/* Phosphor icon path */}
          </svg>
          ```

       Content in Spanish.
  </action>
