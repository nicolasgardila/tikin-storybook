    ## Instalación

    El Design System de Black Cash está construido con React y Tailwind CSS. Para usar los componentes en tu proyecto, asegúrate de tener las siguientes dependencias:

    ```bash title="Terminal"
    npm install tailwindcss@latest react react-dom
    ```

    ## Configuración de Tokens

    Los tokens del DS se definen como CSS custom properties. Agrega las variables al CSS global de tu proyecto:

    ```css title="globals.css"
    :root {
      /* Warm tone */
      --ds-warm-bg: #FAF8F4;
      --ds-warm-surface: #F1EFEB;
      --ds-warm-border: #E8E4DD;

      /* Neutral tone */
      --ds-neutral-bg: #F6F6F6;
      --ds-neutral-surface: #FFFFFF;
      --ds-neutral-border: #DEDEDE;

      /* Brand */
      --ds-brand: #0B0B0B;
      --ds-brand-on: #FFFFFF;
    }
    ```

    ## Tipografía

    El DS usa tres familias tipográficas de Google Fonts:

    | Familia | Rol | Google Fonts |
    |---------|-----|-------------|
    | **Funnel Display** | Títulos y labels | `Funnel+Display` |
    | **Ubuntu Sans** | Texto de cuerpo | `Ubuntu+Sans` |
    | **Ubuntu Sans Mono** | Números y CTAs | `Ubuntu+Sans+Mono` |

    ```html title="index.html"
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link href="https://fonts.googleapis.com/css2?family=Funnel+Display:wght@300..800&family=Ubuntu+Sans:wght@300..700&family=Ubuntu+Sans+Mono:wght@400..700&display=swap" rel="stylesheet">
    ```

    ## Iconos

    Black Cash usa [Phosphor Icons](https://phosphoricons.com) como librería de iconos oficial. Siempre usa SVGs inline, nunca icon fonts.

    ```tsx title="icon-example.tsx"
    // Siempre inline SVG
    <svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 256 256">
      <path d="M128,24A104,104,0,1,0,232,128,104.11,104.11,0,0,0,128,24Z" fill="currentColor"/>
    </svg>
    ```

    ## Uso de Componentes

    Cada componente en esta documentación incluye código copy-paste listo para producción:

    ```tsx title="button-usage.tsx"
    export function PrimaryButton({ children }: { children: React.ReactNode }) {
      return (
        <button className="rounded-xl bg-[var(--ds-brand)] px-6 py-3 text-sm font-medium uppercase tracking-wider text-[var(--ds-brand-on)]">
          {children}
        </button>
      );
    }
    ```

    ## Estructura de la Documentación

    - **Foundations** — Tokens de diseño: colores, tipografía, espaciado, radios, sombras
    - **Components** — 30+ componentes con preview, código y props
    - **Guides** — Guías de instalación, iconos y changelog

    ## Corner Smoothing

    El DS aplica 60% de corner smoothing en todos los radios. Usa el plugin `tailwind-corner-smoothing`:

    ```bash title="Terminal"

    ```

    ```tsx title="smooth-corners.tsx"
    <div className="rounded-xl corner-smooth-60">
      {/* Corner smoothing aplicado */}
    </div>
    ```
    ```
