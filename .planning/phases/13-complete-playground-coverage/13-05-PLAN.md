<task id="1" type="auto">
**Create DS components for Navigation, Chip, and Date Picker**

Create three DS component files following the ds-button.tsx pattern:

1. **components/ds/ds-navigation.tsx** — DSNavigation
   - Props: variant (topbar/tabbar/sidebar)
   - TopBar: TIKIN. Funnel Display 700, 40x40 r12 action buttons
   - TabBar: h64, inner 343x48, FAB 48x44 r12, tab labels Mono 10px
   - Sidebar: w244, bg #FAF8F4, nav labels Funnel Display 13px 400
   - Renders the selected variant with sample navigation items

2. **components/ds/ds-chip.tsx** — DSChip
   - Props: label (text), variant (default/active/disabled), size (sm/md/lg), showClose (boolean)
   - r44 pill, 0.5px border, Ubuntu Sans Mono font
   - Sizes: sm 28px / md 32px / lg 36px height
   - Close icon rendered as small X when showClose is true

3. **components/ds/ds-date-picker.tsx** — DSDatePicker
   - Props: selectedDay (number 1-31), month (text)
   - Double-layer container: outer r24 #F6F6F6 p12, inner r20 #FFFFFF p20
   - Day cells: 36x36 r10 (not circular), weekday Ubuntu Sans Mono 11px
   - Selected day highlighted with brand color

<task id="2" type="auto">
**Create remaining DS components + all playground wrappers + register + add to MDX**

1. Create remaining DS components:

   **components/ds/ds-slider.tsx** — DSSlider
   - Props: value (number 0-100), min (number), max (number), disabled (boolean)
   - Track 4px default, thumb 20px r50% 2px border
   - Disabled color #D9D9D9
   - Renders a static slider visualization at the given value position

   **components/ds/ds-snackbar.tsx** — DSSnackbar
   - Props: message (text), variant (info/success/error/warning), showAction (boolean)
   - Double-layer shell pattern: outer r16 + inner r14
   - Action button: Ubuntu Sans Mono 500 13px uppercase, color #FDDC72 (gold)
   - Dark bg similar to toast pattern

2. Create playground wrappers for all 5 components:

   **navigation-playground.tsx** controls:
   - variant: select, default 'topbar', options ['topbar', 'tabbar', 'sidebar']

   **chip-playground.tsx** controls:
   - label: text, default 'Etiqueta'
   - variant: select, default 'default', options ['default', 'active', 'disabled']
   - size: select, default 'md', options ['sm', 'md', 'lg']
   - showClose: boolean, default false

   **date-picker-playground.tsx** controls:
   - selectedDay: number, default 15, min 1, max 31
   - month: text, default 'Marzo'

   **slider-playground.tsx** controls:
   - value: number, default 50, min 0, max 100
   - min: number, default 0, min 0, max 100
   - max: number, default 100, min 0, max 100
   - disabled: boolean, default false

   **snackbar-playground.tsx** controls:
   - message: text, default 'Operacion exitosa'
   - variant: select, default 'info', options ['info', 'success', 'error', 'warning']
   - showAction: boolean, default false

3. Register all 5 playground components in mdx-components.tsx

4. Add playground as first element after intro in each MDX page:

   - date-picker.mdx: `<DatePickerPlayground />`
