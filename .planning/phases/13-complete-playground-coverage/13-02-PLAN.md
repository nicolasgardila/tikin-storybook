<task id="1" type="auto">
**Create DS components for Search Bar, OTP Input, and Segmented Control**

Create three DS component files following the ds-button.tsx pattern:

1. **components/ds/ds-search-bar.tsx** — DSSearchBar
   - Props: placeholder (text), size (sm/md/lg), disabled (boolean)
   - Renders a search input with icon, r12, 0.5px border, flex layout gap 8px
   - Uses --preview-* CSS vars for theme adaptation

2. **components/ds/ds-otp-input.tsx** — DSOTPInput
   - Props: length (number 4-6), size (xs/sm/md/lg), state (default/active/error/disabled)
   - Renders div boxes (not native inputs) for consistent preview
   - Cell dimensions 48x56 desktop, border 0.5px, error color #EC221F

3. **components/ds/ds-segmented-control.tsx** — DSSegmentedControl
   - Props: tabs (text, comma-separated), activeTab (number), size (sm/md)
   - Ubuntu Sans Mono uppercase, r14 container, p3, warm bg #F1EFEB
   - Active segment: #0B0B0B fill, r12

<task id="2" type="auto">
**Create remaining DS components + all playground wrappers + register + add to MDX**

1. Create remaining DS components:

   **components/ds/ds-avatar.tsx** — DSAvatar
   - Props: size (select 24/32/40/48/64), initials (text), showStatus (boolean), status (online/offline/busy/away)
   - Size-to-font mapping: 24->9, 32->11, 40->14, 48->16, 64->22
   - Status colors: online=#14AE5C, offline=#B3B3B3, busy=#EC221F, away=#E8B931
   - Initials bg: #E8E4DD light, border 0.5px solid #DEDEDE

   **components/ds/ds-tooltip.tsx** — DSTooltip
   - Props: text (text), position (top/bottom/left/right)
   - 13px font, 280px max-width, shadow 0px 2px 8px rgba(11,15,16,0.16)
   - Arrow uses CSS border triangle technique (6px)
   - Renders tooltip always visible with sample trigger element

2. Create playground wrappers for all 5 components:

   **search-bar-playground.tsx** controls:
   - placeholder: text, default 'Buscar...'
   - size: select, default 'md', options ['sm', 'md', 'lg']
   - disabled: boolean, default false

   **otp-input-playground.tsx** controls:
   - length: number, default 4, min 4, max 6
   - size: select, default 'md', options ['xs', 'sm', 'md', 'lg']
   - state: select, default 'default', options ['default', 'active', 'error', 'disabled']

   **segmented-control-playground.tsx** controls:
   - tabs: text, default 'Tab 1, Tab 2, Tab 3'
   - activeTab: number, default 0, min 0, max 4
   - size: select, default 'md', options ['sm', 'md']

   **avatar-playground.tsx** controls:

   - initials: text, default 'NG'
   - showStatus: boolean, default false
   - status: select, default 'online', options ['online', 'offline', 'busy', 'away']

   **tooltip-playground.tsx** controls:
   - text: text, default 'Tooltip content'
   - position: select, default 'top', options ['top', 'bottom', 'left', 'right']
