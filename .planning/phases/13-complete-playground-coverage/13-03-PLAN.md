## Tasks

<task id="1" type="auto">
**Create DS components for Skeleton, Divider, and Progress Bar**

Create three DS component files following the ds-button.tsx pattern:

1. **components/ds/ds-skeleton.tsx** — DSSkeleton
   - Props: variant (text/circle/rectangle), width (text), height (text)
   - Warm bg #EEEBE6 (not #F6F6F6), text variant r4, shimmer animation 1.8s
   - Circle variant uses border-radius 50%
   - Rectangle variant uses r8

2. **components/ds/ds-divider.tsx** — DSDivider
   - Props: variant (neutral/warm/subtle/strong), label (text), showLabel (boolean)
   - Hairline 0.5px, label Ubuntu Sans Mono 11px uppercase
   - 4 color variants with appropriate border colors

3. **components/ds/ds-progress-bar.tsx** — DSProgressBar
   - Props: value (number 0-100), size (sm/md/lg), variant (bar/steps), steps (number)
   - Track: sm=4px, md=6px, lg=8px, bg #EEEBE6, r100px
   - Steps variant: 32px circles with numbers, Funnel Display 12px labels
   - Three-state pattern for steps: completed (filled), active (filled+ring), upcoming (hollow)

**Create remaining DS components + all playground wrappers + register + add to MDX**

1. Create remaining DS components:

   **components/ds/ds-bottom-sheet.tsx** — DSBottomSheet

   - Handle bar: 40x4 r8 #E1E1E1/#434343

   - Title: Funnel Display 400 20px

   **skeleton-playground.tsx** controls:
   - variant: select, default 'text', options ['text', 'circle', 'rectangle']
   - width: text, default '200px'
   - height: text, default '16px'

   **divider-playground.tsx** controls:
   - variant: select, default 'neutral', options ['neutral', 'warm', 'subtle', 'strong']
   - label: text, default 'Section'

   **progress-bar-playground.tsx** controls:
