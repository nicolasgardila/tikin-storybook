## Tasks

<task id="1" type="auto">
**Create DS components for Accordion, Popover, and Table**

Create three DS component files following the ds-button.tsx pattern:

1. **components/ds/ds-accordion.tsx** — DSAccordion
   - Props: items (text, comma-separated), openIndex (number)
   - Funnel Display 15px 400 titles, r20 container, chevron opacity 0.35
   - Renders accordion items with one open showing sample content
   - Uses --preview-* CSS vars for theme adaptation

2. **components/ds/ds-popover.tsx** — DSPopover
   - Props: title (text), showArrow (boolean)
   - Double-layer: outer r20 p8 muted + inner r16 surface
   - Items 10px 14px padding, icons 18px
   - Renders in permanently visible state with sample menu items

3. **components/ds/ds-table.tsx** — DSTable
   - Props: rows (number 3-10), showHeader (boolean)
   - Container r20 with 0.5px border (matches card DS pattern)
   - Headers: Ubuntu Sans Mono 12px uppercase, weight 500, opacity 0.5
   - Generates sample data rows (name, email, role pattern)

<done>
- All three DS component files exist and export their components
- Each component renders with inline styles and --preview-* CSS vars
- Components accept the specified props with sensible defaults
</done>
</task>

<task id="2" type="auto">
**Create remaining DS components + all playground wrappers + register + add to MDX**

1. Create remaining DS components:

   **components/ds/ds-pagination.tsx** — DSPagination
   - Props: totalPages (number), currentPage (number), size (sm/md)
   - Buttons: 32x32 r10 (default), Mono 13px, active bg #0B0B0B
   - Renders page number buttons with prev/next arrows

   **components/ds/ds-breadcrumbs.tsx** — DSBreadcrumbs
   - Props: items (text, comma-separated), separator (select chevron/slash)
   - Current item: Mono 500, separator padding 4px
   - Last item styled as current (bold), others as links

2. Create playground wrappers for all 5 components:

   **accordion-playground.tsx** controls:
   - items: text, default 'Item 1, Item 2, Item 3'
   - openIndex: number, default 0, min -1, max 5

   **popover-playground.tsx** controls:
   - title: text, default 'Opciones'
   - showArrow: boolean, default true

   **table-playground.tsx** controls:
   - rows: number, default 5, min 3, max 10
   - showHeader: boolean, default true

   **pagination-playground.tsx** controls:
   - totalPages: number, default 10, min 1, max 50
   - currentPage: number, default 1, min 1, max 50
   - size: select, default 'md', options ['sm', 'md']

   **breadcrumbs-playground.tsx** controls:
   - items: text, default 'Home, Products, Detail'
   - separator: select, default 'chevron', options ['chevron', 'slash']

3. Register all 5 playground components in mdx-components.tsx

4. Add playground as first element after intro in each MDX page:
   - accordion.mdx: `<AccordionPlayground />`
   - popover.mdx: `<PopoverPlayground />`
   - table.mdx: `<TablePlayground />`
   - pagination.mdx: `<PaginationPlayground />`
   - breadcrumbs.mdx: `<BreadcrumbsPlayground />`

<done>

</task>
