## Tech Debt (Non-Critical)

1. **Prose interference**: fumadocs `.prose` styles override inline-styled elements inside ComponentPreview. Mitigated with `.ds-preview-area` CSS reset but some edge cases may surface with new components.

2. **MDX text wrapping**: MDX processor wraps multiline text between JSX tags in `<p>` elements with margin. Workaround: always inline text content on the same line as the JSX tag.
